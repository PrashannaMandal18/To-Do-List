📝 To-Do List Web App

A simple, modern, and user-friendly To-Do List application designed to help users manage daily tasks efficiently and improve productivity.

🚀 Features
Add new tasks with ease
Delete tasks instantly
Mark tasks as completed
Task timer (start/pause functionality)
Persistent storage using Local Storage
Clean and responsive user interface

🛠️ Built With
HTML – Structure of the application
CSS – Styling and responsive design
JavaScript – Functionality and logic

💡 Overview
This application provides an efficient way to organize daily activities. It ensures tasks are saved even after refreshing the page and offers a smooth, interactive experience for users.
